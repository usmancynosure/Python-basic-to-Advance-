# Python Basic to Advance

This repository contains Python code and Jupyter notebooks covering concepts from basic to advanced, including:
- Functions and functional programming
- Classes and objects
- Encapsulation and abstraction
- Real-world examples

## Structure

- `4-Functions/` — Functional programming examples
- `8-Class And Objects/` — OOP concepts and examples

## Getting Started

1. Clone the repository:
   ```
   git clone https://github.com/usmancynosure/Python-basic-to-Advance-.git
   ```
2. (Optional) Create and activate a virtual environment.
3. Install dependencies as needed.
